 การแนะนำระบบซอฟต์แวร์ Discuz! X
+----------------------------------+
Crossday Discuz! Board（เรียกย่อว่า Discuz!，เลขทะเบียนลิขสิทธิ์จากสำนักบริหารลิขสิทธิ์
แห่งชาติจีน 2008SR11708）เป็นของ บริษัท เทนเซ็นคลาวด์คอมพิวติ้ง (ปักกิ่ง) จำกัด เดิมชื่อ 
บริษัท ปักกิ่งคังเซิงซินฉวงวิทยาศาสตร์และเทคโนโลยี จำกัด (เรียกย่อว่า Comsenz) ซอฟต์แวร์เว็บบอร์ดชุมชน
เป็นเจ้าของโดย Comsenz ในฐานะเป็นผู้ให้บริการซอฟต์แวร์ และผู้ให้บริการชุมชนรายใหญ่ที่สุดใน
ประเทศจีน ผู้นำการให้บริการผลิตภัณฑ์ Discuz! และ ผลิตภัณฑ์ที่คล้ายกันทั้งในและต่างประเทศ
ไม่ว่าในแง่ของการทำงาน เสถียรภาพ ความจุโหลด การรับประกันความปลอดภัย ฯลฯ

เป็นหนึ่งในระบบซอฟต์แวร์เว็บบอร์ดที่มีวุฒิภาวะสูงสุดและครอบคลุมมากที่สุดในโลก
ปัจจุบันดูแลโดย บริษัท เหอเฟย์เน็ตเวิร์คเทคโนโลยี จำกัด
